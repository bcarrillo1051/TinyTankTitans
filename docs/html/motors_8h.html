◆ task_barrel()

void task_barrel ( void * p_param)

Task controls the Tank's Barrel.

This task runs a FSM which controls the Barrel of the tank. There are 3 possible states: Idle, Up, and Down. The state depends on the data sent via bluetooth.

Parameters
p_paramsA pointer to parameters passed to this task. This pointer is ignored; it should be set to NULL in the call to xTaskCreate() which starts this task

◆ task_movement()

void task_movement ( void * p_param)

Task controls 2D motion of the Tank.

This task runs a FSM which controls the 2D motion of the tank. There are 5 possible states: Idle, Forward, Backwards, Left, Right. The state depends on the data sent via bluetooth.

Parameters
p_paramsA pointer to parameters passed to this task. This pointer is ignored; it should be set to NULL in the call to xTaskCreate() which starts this task
